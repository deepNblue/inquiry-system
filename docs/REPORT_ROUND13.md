## 核心功能

### 1. 邮件收取模块
- 支持 IMAP/POP3 协议
- 自动提取邮件中的价格
- 关联原始询价

### 2. 置信度体系
- **来源权重**: 官网100% > 京东95% > 淘宝85% > 邮件60% > 历史50%
- **时效衰减**: 每月-5%
- **参数匹配**: 品牌/型号/规格匹配

### 3. 参数偏离检测
- 智能对比需求与实际规格
- 自动标注偏离情况
- 严重程度分级 (critical/warning/info)
- 生成对比报告

### 4. 增强结果聚合
- 整合网页/厂家/邮件/历史四种来源
- 计算整体置信度
- 判断是否符合要求
- 生成Markdown报告
