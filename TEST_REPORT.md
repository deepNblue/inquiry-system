## 🔍 发现的问题

### 1. 网页抓取受限

**现象**: 京东/淘宝/拼多多等主流电商都有严格反爬

**原因**:
- UA 检测
- 请求频率限制
- JS 渲染内容
- 验证码

**解决方案**:
1. 配置 Firecrawl API (推荐)
2. 使用代理池
3. 降低请求频率

### 2. 价格解析 Bug

**现象**: "原价 ¥14999 现价 ¥12999" 解析为 ¥14999

**原因**: 正则只匹配价格，无法区分上下文

**建议**: 使用 LLM 理解上下文
